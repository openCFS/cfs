## Description

This disk image file contains the  nightly built CFS++ OSX binaries for @ARCH@
from @DATE@.  Since the `.dmg` format  has problems with some  file names, the
CFS++ build directory  has been zipped. Please follow  the instructions in the
next sections to start using CFS++.


## Getting Started

### Extract Archive

First you  need to start the  OSX Terminal application.  Then  you can extract
the `.zip` archive  to your desired location and start  using CFS++ from there
in the follwing way:

~~~~ {.bash}
# Make a temporary directory
cd $HOME
mkdir cfs && cd cfs

# Extract zip
unzip /Volumes/CFS_NIGHTLY_@ARCH@_@DATE@/CFS_BUILD_NIGHTLY.zip

# Start CFS++ and print out some infos about the executable
CFS_BUILD_NIGHTLY/bin/cfs -v
~~~~

### Starting your First Simulation

In the  directory `/Volumes/CFS_NIGHTLY_@DATE@/example` you can  find a simple
example for an electro-static field simulation. To run the simulation copy the
contents of this  folder to your temp directory and  start the simulation from
there:

~~~~ {.bash}
# Change into temporary directory
cd $HOME/cfs

# Copy example folder
cp -Rv /Volumes/CFS_NIGHTLY_@ARCH@_@DATE@/example .

# Change into example folder and make it writable
cd example && chmod +w . *

# Start simulation
../CFS_BUILD_NIGHTLY/bin/cfs example
~~~~

### Looking at the Results

The  results of  the simulation  are written  in Gmsh  format into  the folder
`results_gmsh` by CFS++. Gmsh is a  pre- and postprocessor for FEM. You should
go to the [download section of the Gmsh webpage](http://geuz.org/gmsh) and get
Gmsh for OSX there. Once you have installed Gmsh, you can just open the result
file generated by CFS++.
